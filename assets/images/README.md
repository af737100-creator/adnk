# مجلد الصور

هذا المجلد يحتوي على صور المنصة. يمكنك إضافة الصور التالية:

## الصور المطلوبة:

1. `hero-dashboard.png` - صورة العرض الرئيسي
2. `default-thumbnail.jpg` - صورة افتراضية للفيديوهات
3. `logo.png` - شعار المنصة
4. `favicon.ico` - أيقونة الموقع

## صور توضيحية:

5. `feature-1.png` - ميزة الأسئلة التفاعلية
6. `feature-2.png` - ميزة الاختبارات
7. `feature-3.png` - ميزة الشروحات الإضافية
8. `how-it-works.png` - شرح كيفية عمل المنصة

يمكنك استخدام صور بديلة أو إنشاء صورك الخاصة.
