# مجلد الخطوط - تعليمي

هذا المجلد مخصص لإضافة الخطوط المخصصة للمنصة.

## الخطوط المستخدمة في المنصة

حالياً، تستخدم المنصة الخطوط التالية من Google Fonts:

### الخطوط العربية
- **Cairo** - للعناوين والنصوص الرئيسية
- **Tajawal** - للنصوص الطويلة والمحتوى
- **Almarai** - بديل للخطوط العربية

### الخطوط اللاتينية
- **Inter** - للواجهات والنصوص التقنية
- **Poppins** - للعناوين
- **Roboto** - للنصوص العامة
