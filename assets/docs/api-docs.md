# توثيق API - تعليمي

## نظرة عامة

يوفر تعليمي واجهة برمجة تطبيقات (API) للمطورين لدمج ميزات المنصة في تطبيقاتهم الخارجية. هذا التوثيق يشرح كيفية استخدام هذه الواجهات.

## المصادقة

### الحصول على مفتاح API

1. سجل الدخول إلى حسابك في تعليمي
2. اذهب إلى الإعدادات ← التكاملات ← API
3. انقر على "إنشاء مفتاح API جديد"
4. انسخ المفتاح واحفظه في مكان آمن

### استخدام المفتاح

أضف المفتاح في ترويسة الطلب:

```http
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
