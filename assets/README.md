# مجلد الأيقونات

هذا المجلد يمكن أن يحتوي على أيقونات مخصصة للمنصة.

## الأيقونات المطلوبة:

1. `favicon.ico` - أيقونة الموقع
2. `apple-touch-icon.png` - أيقونة iOS
3. `icon-192.png` - أيقونة PWA 192x192
4. `icon-512.png` - أيقونة PWA 512x512

## لإنشاء أيقونات:

يمكنك استخدام أدوات مثل:
- [Favicon Generator](https://favicon.io)
- [RealFaviconGenerator](https://realfavicongenerator.net)

## ملاحظة:

نحن نستخدم Font Awesome للأيقونات في هذا المشروع، لذلك ليست هناك حاجة لأيقونات إضافية إلا للأغراض المذكورة أعلاه.
