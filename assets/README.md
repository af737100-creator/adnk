# تعليمي - منصة التعلم التفاعلي

منصة تعليمية تسمح بتحويل مقاطع اليوتيوب التعليمية إلى فصول تفاعلية مع إضافة أسئلة، اختبارات قصيرة، وشروحات إضافية.

## المميزات الرئيسية

1. **تحويل فيديوهات اليوتيوب إلى دروس تفاعلية**
   - أضف أسئلة في أي نقطة من الفيديو
   - اختبارات قصيرة متعددة الخيارات
   - شروحات إضافية عند نقاط معينة

2. **لوحة تحكم متكاملة**
   - إدارة الدورات التعليمية
   - متابعة تقدم الطلاب
   - تحليلات وأحصاءات مفصلة

3. **غرف تعليمية تفاعلية**
   - سبورة تفاعلية مشتركة
   - دردشة في الوقت الحقيقي
   - ميزة رفع الأيدي

4. **مميزات تنافسية**
   - مجانية تمامًا للمعلمين
   - دعم كامل للغة العربية
   - تصدير إلى أنظمة إدارة التعلم (LMS)
   - تصميم متجاوب يعمل على جميع الأجهزة

## التكنولوجيا المستخدمة

- **واجهة أمامية:** HTML5, CSS3, JavaScript
- **قاعدة البيانات:** Supabase (PostgreSQL)
- **المصادقة:** Supabase Auth
- **الاستضافة:** GitHub Pages, Vercel
- **التفعيل في الوقت الحقيقي:** Supabase Realtime
